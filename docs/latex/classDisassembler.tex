\doxysubsection{Detailed Description}
A disassembler turning 2 bytes chip8 instructions into human readable strings. 

This class stores all functionality required to turn a 2 byte chip8 instruction into human readable string. It has only one public method -\/ \mbox{\hyperlink{classDisassembler_a2de7da3bc8cc0543655fae1879e201c6}{Disassembler\+::disassemble}} -\/ which returns disassembly of provided instruction.

The returned strings conform to \href{http://devernay.free.fr/hacks/chip8/C8TECH10.HTM}{\texttt{ Cowgod\textquotesingle{}s Chip-\/8 Technical Reference v1.\+0}}
