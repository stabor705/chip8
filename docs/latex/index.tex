\begin{DoxyItemize}
\item Renders CHIP-\/8 games using Open\+GL.
\item Displays disassembly of processed instructions along with current chip8 state.
\item Allows to run games in instruction-\/by-\/instruction manner.
\item Controlled by clicking on displayed keypad with 16 keys.
\item Can load files passed as program arguments and selected using interactive file dialog.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{Requirements}\label{index_autotoc_md3}
